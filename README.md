# AI Data Agent (SQL + Insights)

## Setup

1. Install dependencies:
```
pip install -r requirements.txt
```

2. Install Ollama and run:
```
ollama run mistral
```

3. Create database:
```
python database.py
```

4. Run agent:
```
python main.py
```

## Example Queries
- Top products by revenue
- Total revenue in India
- Which region has highest sales?
